# Plexo UI Kit

Plexo — мобильное приложение для сканирования вещей, ведения профиля и торговли. Этот репозиторий содержит живой прототип фронтенда и ключевые спецификации для handoff командe разработки.

Радиусы: 24 px для крупных карточек, 16 px для кнопок/inputs, 50% для круглых элементов. Сетка базируется на 8‑пиксельном шаге.

## Компоненты

- **Auth / Onboarding**: логотип, загрузка аватара, поля (Inter, 14–24 px).
- **Bottom Nav**: 5 иконок (Настройки, Рынок, Сканер, Чаты, Профиль).
- **Scanner**: полноэкранная камера, кнопка съемки, подсказки, модальное окно анализа.
- **Profile**: header + rating + grid (3 в ряд), модалка карточки.
- **Market**: поиск, фильтры, карточки 4:3, детальное окно продавца.
- **Chat**: список сообщений, input с вложениями, статусы прочтения.
- **Settings**: список секций (Профиль, Уведомления, Тема, Безопасность, Помощь, Выйти).
- **Modals**: подтверждение удаления, смена пароля, upload.

## Анимации и UX

- Переходы между экранами — slide/push 250–300 ms ease-out.
- Всплывающие карточки — translateY + opacity 0.3 s.
- Кнопка сканера нажата — scale 0.96 + тень усиливается.
- Успешное добавление вещи — чек + bounce microinteration.
- Ошибка сканирования — shake карточки + подсказка (aria-live).

Для доступности соблюдаем контраст ≥4.5:1 для заголовков, ≥3:1 для body, кликабельные зоны ≥44 px. Все иконки имеют подписи/aria-label.

## Handoff

1. Figma файл с компонентной системой (цвета, типографика, компоненты, nav, состояние).
2. Экспорт SVG/PNG (логотипы, иконки) + WebP/AVIF для фотографий.
3. Design tokens (JSON) c палитрой, spacing и radius.
4. Lottie/MP4 с примерами микровзаимодействий (сканер, чекбокс, ошибки).

## Нейросетевой UX

- Для обработки >2–3 секунд показывать прогрессбар + подсказки.
- Confidence < 60% → выводим предупреждение, просим ручную корректировку.
- Показываем историю товара + «проверено» после модерации.
